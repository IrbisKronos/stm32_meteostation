# STM32 MeteoStation

Метеостанція на базі STM32F411CEU6 (Black Pill), яка вимірює температуру, вологість і атмосферний тиск за допомогою датчика BME280 та відображає дані на OLED-дисплеї SSD1306.

## Можливості

- Вимірювання температури (°C), відносної вологості (%) та атмосферного тиску (hPa) кожні 2 секунди
- Відображення даних на монохромному OLED-дисплеї 128×64 пікселів
- Паралельний вивід даних через UART (115200 бод) для відладки в терміналі
- Heartbeat-індикація на вбудованому LED (PC13)

## Апаратне забезпечення

### Мікроконтролер

| Параметр        | Значення                   |
| --------------- | -------------------------- |
| МК              | STM32F411CEU6 (Black Pill) |
| Архітектура     | ARM Cortex-M4              |
| Тактова частота | 100 МГц (HSI → PLL)        |
| Flash           | 512 КБ                     |
| RAM             | 128 КБ                     |
| Корпус          | UFQFPN48                   |

### Підключені компоненти

| Компонент                                   | Інтерфейс | Адреса |
| ------------------------------------------- | --------- | ------ |
| BME280 (датчик температури/вологості/тиску) | I2C1      | 0x76   |
| SSD1306 (OLED 128×64)                       | I2C1      | 0x3C   |

### Розпіновка

| Пін МК | Функція     | Пристрій                   |
| ------ | ----------- | -------------------------- |
| PB6    | I2C1 SCL    | BME280 + SSD1306           |
| PB7    | I2C1 SDA    | BME280 + SSD1306           |
| PA9    | USART1 TX   | UART (вивід у термінал)    |
| PA10   | USART1 RX   | UART                       |
| PC13   | GPIO Output | Вбудований LED (heartbeat) |
| PA13   | SWD SWDIO   | Програматор ST-Link        |
| PA14   | SWD SWCLK   | Програматор ST-Link        |

### Схема підключення I2C

```
STM32F411 (Black Pill)
       PB6 (SCL) ──┬──── BME280 SCL
                   └──── SSD1306 SCL
       PB7 (SDA) ──┬──── BME280 SDA
                   └──── SSD1306 SDA
       3.3V ────────┬──── BME280 VCC
                   └──── SSD1306 VCC
       GND ─────────┬──── BME280 GND
                   └──── SSD1306 GND
```

> Обидва пристрої на одній I2C-шині, підтяжні резистори (4.7 кОм) до 3.3V на лініях SCL/SDA.

## Програмне забезпечення

### Структура проєкту

```
CubeMeteostation/
├── Core/
│   ├── Inc/
│   │   ├── main.h          — визначення пінів та загальні заголовки
│   │   ├── bme280.h        — API драйвера BME280
│   │   ├── ssd1306.h       — API драйвера SSD1306
│   │   └── fonts.h         — шрифти для відображення тексту
│   └── Src/
│       ├── main.c          — головна логіка програми
│       ├── bme280.c        — драйвер датчика BME280
│       └── ssd1306.c       — драйвер OLED-дисплея SSD1306
├── Drivers/
│   ├── CMSIS/              — ARM CMSIS та заголовки STM32F4
│   └── STM32F4xx_HAL_Driver/ — ST HAL бібліотека
├── CubeMeteostation.ioc    — конфігурація STM32CubeMX
├── STM32F411CEUX_FLASH.ld  — скрипт лінкера (Flash)
└── STM32F411CEUX_RAM.ld    — скрипт лінкера (RAM)
```

### Логіка роботи

1. Ініціалізація периферії (I2C1, USART1, GPIO)
2. Ініціалізація SSD1306 та BME280 з діагностикою через UART
3. Відображення стартового екрану "MeteoStation / Starting..."
4. Головний цикл (кожні 2 секунди):
   - Зчитування даних з BME280 (температура, вологість, тиск)
   - Оновлення OLED-дисплея
   - Вивід даних у UART
   - Перемикання LED (heartbeat)

### Формат виводу на OLED

```
┌──────────────────────────────┐  ← жовта зона (16 px)
│       MeteoStation           │
├──────────────────────────────┤
│ Temp:  23.5 C                │
│ Humid: 48.2 %                │  ← синя зона (48 px)
│ Press: 1013 hPa              │
├──────────────────────────────┤
│ UART:OK  SD:--               │
└──────────────────────────────┘
```

### Формат UART-логу

```
T=23.52 C  H=48.2 %  P=1013.2 hPa
```

## Збірка та прошивка

### Вимоги

- [STM32CubeIDE 2.1.1](https://www.st.com/en/development-tools/stm32cubeide.html) або новіша
- ST-Link програматор (вбудований на Nucleo або зовнішній)

### Збірка

1. Відкрити STM32CubeIDE
2. `File → Open Projects from File System` → вибрати папку репозиторію
3. `Project → Build Project` (Ctrl+B)

### Прошивка

1. Підключити плату через ST-Link
2. `Run → Debug` (F11) або `Run → Run` (Ctrl+F11)

### Моніторинг UART

Підключитися до USART1 (PA9 TX) будь-яким термінальним емулятором:

| Параметр  | Значення |
| --------- | -------- |
| Baudrate  | 115200   |
| Data bits | 8        |
| Parity    | None     |
| Stop bits | 1        |
